Convertor valutar, foloseste cursul oficial al BNR.
== Description ==
Plugin-ul adauga in sidebar un convertor valutar bazat pe cursul oficial al BNR pentru ziua curenta. Este disponibila si traducerea in limba engleza a plugin-ului.



== Installation ==

* copiati folderul "convertor-valutar" in directorul /wp-content/plugins al blogului
* activati plugin-ul
* accesati meniul Appearance ==> Widgets
* adaugati pluginul "Convertor valutar" in sidebar
* setati titlul ce doriti sa fie afisat pt. widget si salvati setarile
